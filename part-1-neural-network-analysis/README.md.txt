# Customer Churn Prediction using Neural Network

## Project Overview
This project focuses on predicting customer churn using a Feed-Forward Neural Network built with TensorFlow/Keras.

The dataset contains customer demographic and service usage information. The objective is to classify whether a customer is likely to churn or not.

---

## Tasks Performed

### Task 1: Dataset Understanding
- Dataset exploration
- Missing value analysis
- Statistical summary
- Target variable distribution

### Task 2: Data Preprocessing
- Missing value handling
- Label Encoding
- Feature Scaling
- Train-Test Split

### Task 3: Neural Network Model Building
- Feed-forward neural network
- Hidden layers with activation functions
- Binary classification output layer

### Task 4: Model Evaluation
- Accuracy evaluation
- Loss analysis
- Prediction testing

### Task 5: Hyperparameter Experimentation
- Different hidden layers
- Different neurons
- Learning rate changes
- Epoch and batch size tuning

### Task 6: Final Reflection
- Weights and biases
- Activation functions
- Learning rate analysis
- Underfitting and overfitting discussion
